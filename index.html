Playstation Trophy Collection Site

This site uses Google Firebase to allow users to create an account. Users can take a PlayStation platinum trophy guide from PSN profiles and input the link into a box. The site then parses that link for all the trophies and their descriptions, displaying them to the user. Users have the option to store the list to their account, where they can view every list they've saved.

Video Game Backlog Tracker

This program uses parsing to retrieve data from the video game website HowLongToBeat. Specifically, for any game link the user inputs, it fetches data on how long it takes to beat the main story, main + sides, completionist, and all styles. Once the user finishes, all this data is compiled into an Excel sheet for easy viewing. Additionally, there are columns that aggregate the numbers for each category, allowing users to see the total time needed to complete every style for all their games in the list. This program is designed to help users organize their entire gaming backlog conveniently. Furthermore, it provides average completion times for each game in various styles, reducing the need for users to repeatedly check HowLongToBeat for this information.
